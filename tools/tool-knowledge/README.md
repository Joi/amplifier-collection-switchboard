# tool-knowledge

Knowledge vault operations and gap detection.

## Status: Complete

## Functionality

- Gap detection (undefined concepts, stale content, orphans)
- Citation verification with academic sources
- Staged content approval workflow
- Integration with chanoyu vault

## Commands

- `gaps` - Run gap detection
- `verify` - Verify citations
- `stale` - Find stale content
- `orphans` - Find orphan pages
- `stage` - Stage content for review

## Academic Source Integration

- Semantic Scholar API
- arXiv API
- Tavily search
